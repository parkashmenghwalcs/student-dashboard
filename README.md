# 📊 Student Performance Dashboard

This project is a Data Analytics Dashboard built using Python and Streamlit.

## 🚀 Features
- Student data visualization
- Bar charts and pie charts
- Interactive filtering
- CSV data analysis

## 🛠 Tech Stack
- Python
- Streamlit
- Pandas
- Matplotlib
- Seaborn

## ▶️ How to Run
```bash
streamlit run app.py
